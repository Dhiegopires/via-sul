# Step 2.1 — One-Time Setup Checklist
> Human checklist. Run once per new client, after `0-briefing/0.1-client-intake-checklist.md` has cleared. Estimated time: 3-5 business days. Price: $199-299 (see root README for context on this being intro pricing).

---

## Core information

```
[ ] Name: exact legal/signage name. Zero keywords appended. ("ABC Plumbing" — not
    "ABC Plumbing - Best Emergency Plumber [City]"; the latter is a documented
    suspension trigger, see 4-risk-protocol.md)
[ ] Address: identical to the client's own site and every directory — including
    formatting ("St" vs "Street", "Suite" vs "#"). NAP mismatches are a confirmed
    ranking factor (position #15 in Whitespark's individually-confirmed factors).
[ ] Phone: correct area code, identical everywhere
[ ] Website: linked, no redirect loops or redirect-tracking URLs (a bare redirect
    URL is itself a documented suspension trigger)
[ ] Hours: every day of the week, plus upcoming holidays.
    THIS MATTERS MORE THAN IT LOOKS: "business open at time of search" is
    Whitespark's confirmed #5 ranking factor — a business can rank #1 open and
    drop to #4 the moment it closes. Incomplete or wrong hours cost ranking, not
    just customer trust.
[ ] Primary category: the MOST SPECIFIC option available.
    Wrong: "Contractor". Right: "Plumber" or "Emergency Plumber".
    This is the single most important controllable ranking factor — confirmed
    across every source checked. Use a browser extension like GMB Everywhere to
    see what categories the top 3 competitors in the Map Pack are using.
[ ] Secondary categories: up to 9, every one genuinely applicable — don't pad
[ ] Description: 750 characters max, natural keywords in the first 250.
    Structure: who they are + what they do + where + proof + call to action.
    Example: "Austin's emergency plumber since 2008. We fix leaks, clogs, and
    water heaters fast, with same-day service. Licensed and insured. Call 24/7."
[ ] Attributes: every one genuinely applicable (wheelchair accessible,
    women-owned, veteran-owned, etc.)
[ ] Service area (Service-Area Businesses only — no public storefront):
    - Address must be HIDDEN, per Google's own guidance
      (support.google.com/business/answer/3038177)
    - CONFIRMED TRADEOFF: Sterling Sky ran a real case study (hiding the address
      caused a "massive decline in rankings" until it was restored) — hiding the
      address is required for SABs with no public storefront, but expect a
      ranking hit and plan for it, don't be surprised by it
    - Up to 20 service areas can be defined (confirmed Google limit)
```

## Services

```
[ ] Full list, specific names, natural keywords
[ ] Short description per service (optional, recommended)
[ ] Add Google's own predefined services where they exist for this category
```

## Photos (minimum 10, real photos, not stock)

```
[ ] Logo: minimum 250x250px, transparent or solid background
[ ] Cover photo: 16:9, high resolution, actually representative of the business
[ ] Exterior: minimum 3 angles (unless SAB with hidden address)
[ ] Interior, if applicable
[ ] Team or work in progress
[ ] Products/services being performed

Why this matters more than it sounds like it should: BrightLocal's original
2017-18 photo study (45,264 listings — the exact figures are from that study,
not fresh 2026 data, but the direction is still cited as current best practice)
found profiles with 100+ photos got 520% more calls, 2,717% more direction
requests, and 1,065% more website clicks than sparse profiles. The percentages
are old; the underlying point — a photo-poor profile is invisible, a photo-rich
one gets acted on — still holds.
```

## Q&A

```
[ ] 10-15 common questions pre-populated by the owner (anyone can post a question
    publicly — get ahead of it before a competitor or troll posts a bad one)
```

## Verification

```
[ ] Profile verified (video verification is the standard method for new profiles
    in 2026; postcard has been discontinued for most Service-Area Businesses)
[ ] Manager access configured — your agency account as Manager, client as Owner,
    never the reverse (see 4-risk-protocol.md)
```

## Citations (NAP consistency across the web)

```
[ ] Audit NAP on: Google, Yelp, Bing Places, Apple Maps, BBB, Facebook, Yellow Pages
[ ] Fix every inconsistency found
[ ] Submit to any niche-relevant directory missing (Angi/HomeAdvisor/Houzz for
    home services, Healthgrades for health, Avvo/FindLaw for legal)
```

> - [ ] Categoria primária é a mais específica possível (comparada com os 3 primeiros do Map Pack)
> - [ ] Horários completos, incluindo feriados dos próximos 60 dias
> - [ ] NAP idêntico no site do cliente e no perfil
> - [ ] Mínimo 10 fotos reais publicadas
> - [ ] Sua conta é Manager, não Owner
>
