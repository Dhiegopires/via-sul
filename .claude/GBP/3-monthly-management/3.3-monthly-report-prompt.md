# Step 3.3 — Monthly Client Report
> AI prompt. Paste real GBP Insights data in — never estimate or round up a number the client will see.

---

## Data to pull first (GBP Insights, or your rank-tracking tool if using one)

- Calls, direction requests, website clicks, photo views — this month vs. last month
- New reviews received, current average rating, response rate/time
- Search impressions if available (Direct vs. Discovery vs. Branded)
- Geo-grid scan result if running one (see `../5-measurement/5.1-baseline-and-geogrid.md`)

---

## Prompt

```
Write a one-page monthly Google Business Profile report for [CLIENT NAME], a
[NICHE] in [CITY]. This is a small-business owner, not a marketer — no jargon,
lead with the business outcome, not the metric name.

Structure:
1. Headline: the one number that matters most this month, framed as an outcome
   ("You got 47 calls from Google this month, up 22% from last month") — not
   a jargon metric name
2. What we did this month: [X posts published, Y photos added, Z reviews
   responded to, any structural fixes made]
3. Reviews: new count, current rating, confirm all responded within 48h
4. If anything dropped: name the likely cause in plain language (e.g., "fewer
   direction requests this month, likely because two competitors added new
   photos — we're adding N new ones this month to catch back up"), don't just
   show the number and go silent on it
5. Next month: ONE specific planned action, not a list

Data:
[paste the real numbers pulled above]

Never invent or round a number. If a data point is missing, write "not
available this month" rather than guessing.
```
